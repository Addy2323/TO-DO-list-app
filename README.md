# 📝 React TO-DO List App

A simple and modern to-do list app built with **React** and **Material-UI**.

## 🚀 Features

- Add tasks
- Edit tasks
- Delete tasks
- Simple and responsive UI


## 🧰 Tech Stack

- React
- Material-UI (MUI)

## 🔧 How to Run Locally

```bash
git clone https://github.com/Addy2323/TO-DO-list-app.git
cd TO-DO-list-app
npm install
npm start
